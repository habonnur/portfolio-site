# Habon Nur Portfolio

A responsive static portfolio built with plain HTML and CSS. No build step is required.

## Preview locally

From this directory, run:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Before publishing

1. Confirm that `assets/human-headline.jpg` is licensed for public website use.
2. Replace `og-image.png` if it does not match the final design.
3. Add licensed webfonts if exact cross-platform typography is required.

## GitHub Pages

Push the site to a public GitHub repository. In **Settings → Pages**, choose
**Deploy from a branch**, select `main`, choose `/(root)`, and save.
